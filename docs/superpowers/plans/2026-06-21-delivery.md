# Delivery + Driver Role — Implementation Plan

> Implement task-by-task. No test runner — the gate for every task is `npx tsc --noEmit` (plus `next build` / a screenshot where noted). Commit per task.

**Goal:** Order delivery with zoned fees, a deliveries view, and a `driver` role — admin-side.

**Spec:** `docs/superpowers/specs/2026-06-21-delivery-design.md`.

## Global Constraints
- Hebrew/RTL only; copy in `src/lib/i18n.ts`. DOCKET tokens. `zod/v4`.
- `group_id`-scoped; owner/manager create orders; **driver manages nothing**.
- Migrations auto-run on deploy; hand-write SQL + journal entry (drizzle-kit snapshots are already drifted). Enum `ADD VALUE` gets its own migration file.
- Effective order total = `(totalOverride ?? Σ items) + delivery_fee`.
- Security: every server `role === 'baker'` management deny also denies `driver`; client `isAdmin` = owner||manager.

## File structure
**Create:** `src/lib/delivery.ts` (normalizeCity, suggestDelivery, resolveDeliveryFee, buildWazeLink); `src/app/api/deliveries/route.ts`; `src/app/miniapp/deliveries/page.tsx`; `src/components/delivery/DeliveryRow.tsx`; `drizzle/0014_delivery.sql`, `drizzle/0015_driver_role.sql`.
**Modify:** `src/db/schema.ts`; `src/hooks/useGroup.ts`; `src/lib/order-payments.ts`; `src/app/api/orders/route.ts` + `[id]/route.ts` + `[id]/status/route.ts` + `[id]/pay/route.ts`; `src/app/miniapp/orders/new/page.tsx`; `src/app/api/customers/route.ts` + `[id]/route.ts`; `src/app/miniapp/customers/[id]/page.tsx`; `src/components/ui/DocketRow.tsx`; `src/components/ui/BottomNav.tsx`; `src/app/miniapp/page.tsx`; `src/app/miniapp/layout.tsx`; `src/app/miniapp/settings/page.tsx`; `src/app/api/groups/[id]/route.ts` + `/invites/route.ts`; `src/lib/public-site.ts` + `src/components/public/DetailsSection.tsx`; `src/lib/i18n.ts`; the `=== 'baker'` server sweep; `src/app/miniapp/customers/page.tsx` + `[id]` `isAdmin`.
